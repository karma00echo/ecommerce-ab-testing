# 📊 电商落地页 A/B Test 分析

## 项目背景
某电商平台对落地页进行改版（V1），通过 A/B Test 评估
新版页面是否能有效提升用户转化率。

## 数据集
- 来源：Kaggle Ecommerce AB Testing Dataset
- 规模：200 万条用户行为记录，9 个字段
- 实验周期：2021-02-01 至 2021-02-16（共 16 天）
- 实验组 / 对照组各 100 万用户

## 分析流程
| 步骤 | 内容 |
|------|------|
| 1. 数据读取与清洗 | 缺失值检查、重复值处理、字段验证 |
| 2. EDA 探索分析 | 转化率分布、各维度对比、时间趋势 |
| 3. 核心 A/B Test | 转化率计算、Uplift 测量 |
| 4. 显著性检验 | Z-test、P值、置信区间、功效分析 |
| 5. 分层分析 | 设备、地区、渠道、用户类型异质性分析 |
| 6. 商业洞察 | 业务建议、上线策略、风险提示 |

## 核心结论
| 指标 | 数值 |
|------|------|
| 对照组转化率 | 10.01% |
| 实验组转化率 | 10.99% |
| Uplift | +9.78% |
| P 值 | ≈ 0（p < 0.05）|
| 统计显著性 | ✅ 显著 |

**结论：新版落地页转化率提升显著，建议推进上线。**
